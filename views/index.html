Gerry Quotes

REST API per frasi di Gerry Scotti. Lista degli endpoints:

GaaS: Gerry as a Service

Endpoints

Frase random

GET https://gerry-quotes.fly.dev/quotes/random

Ritorna una frase random tra le disponibili

Esempio di risposta:

{
  "quote": "Con tutti i programmi quotidiani che ho condotto, ho scoperto di aver lavorato il doppio dei giorni richiesti. Tecnicamente è come se facessi questo mestiere da 80 anni."
}

Tutte le frasi
